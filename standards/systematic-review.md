# Systematic Review Standard
Source: ACM SIGSOFT Empirical Standards, CC0.

## Essential Attributes
- Presents a step-by-step, replicable description of the search process
- Defines clear inclusion and exclusion criteria
- Describes the sampling strategy
- Mitigates sampling and publication bias using multiple approaches, such as:
  keyword search, backward/forward snowballing, checking prolific author profiles,
  searching multiple databases, searching preprint servers, soliciting unpublished work
- Defines a data extraction scheme and describes it or provides it as supplementary material
- Synthesizes results — does NOT merely list or summarize studies individually
- Acknowledges generalizability threats; discusses how reviewed studies may differ
  from the target population

## Desirable Attributes
- Provides supplementary materials
  (protocol, search terms, search results, selection process, extraction scheme,
  coding examples, complete dataset, analysis scripts)
- Uses 2+ independent reviewers; reports inter-rater reliability;
  explains how discrepancies were resolved
- Assesses quality of primary studies using an a priori scheme
- Consolidates results using tables, diagrams, or charts
  (PRISMA flow diagram recommended)
- Integrates results into prior theory; identifies gaps and future directions
- Presents practical, evidence-based guidelines for practitioners or researchers

## Invalid Criticisms
- Studies were not published in peer-reviewed venues
- The original studies do not employ a common research design
- The primary studies are not a random sample of the phenomenon
